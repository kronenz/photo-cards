# Design System Components

**Feature**: 003-navigation-ui-renewal
**Purpose**: Reusable UI primitives based on Apple Design System

## Components

- **Button.svelte**: Primary, secondary, ghost button variants
- **Input.svelte**: Text input with validation states
- **Modal.svelte**: Dialog modal with backdrop
- **Card.svelte**: Container with shadow and hover states
- **Spinner.svelte**: Loading indicator
- **SkeletonLoader.svelte**: Content placeholder
- **ToastNotification.svelte**: Temporary success/error messages
- **DropdownMenu.svelte**: Menu with Bits UI headless component
- **Accordion.svelte**: Collapsible sections

All components use design tokens from `src/lib/styles/design-tokens.css`.
