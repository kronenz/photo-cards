# Collections Page Components

**Feature**: 003-navigation-ui-renewal (US3)
**Purpose**: Apple Photos-style collection browser

## Components

- **CollectionGrid.svelte**: Masonry grid layout
- **CollectionCard.svelte**: Individual collection preview
- **CollectionFilters.svelte**: Filter by team, rarity, season
- **SortMenu.svelte**: Sort options (date, name, rarity)

## Features

- Virtual scrolling for performance (60fps guaranteed)
- Keyboard navigation (arrow keys, Enter)
- Multi-select mode (Shift+Click)
